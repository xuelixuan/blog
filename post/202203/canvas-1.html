fill与stroke的区别：fill是填充，stroke是描边

fillStyle是设置填充的颜色，渐变，strokeStyle是设置描边的颜色，渐变

fillText, strokeText分别是设置需要填充的文字和需要描边的文字

            let canvas = document.getElementById("canvas");
            ctx = canvas.getContext("2d");
            ctx.font = "24px Arial";
            ctx.fillStyle = "red"; // 填充canvas的颜色
            ctx.strokeStyle = "blue"; // 笔触的颜色
            ctx.fillText(
                "hello word",
                canvas.width / 2 - 150,
                canvas.height / 2 + 15
            );
            ctx.strokeText(
                "hello word",
                canvas.width / 2 - 100,
                canvas.height / 2
            );
