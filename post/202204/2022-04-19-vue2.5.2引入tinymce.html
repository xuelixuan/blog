vue2.5.2 引入 tinymce

第一步,安装依赖
在package.json 的dependencies 中加入如下配置
"@tinymce/tinymce-vue": "^3.2.8",
"tinymce": "^5.0.11",
然后执行npm install

第二步

安装完成后,进入node_modules/找到tinymce文件夹,整体拷贝到static目录下

第三步

新安装的 tinymce 没有中文本,需要在static/tinymce/langs/下新建目录zh_CN.js文件

zh_CN

第五步,自定义按钮以及图标配置

图标配置:http://tinymce.ax-z.cn/configure/editor-appearance.php#icons 

tinymce.init({
	selector: '#iconsdemo',
	icons: 'custom', // baseURL/icons/custom/icons.js
})

// baseURL/icons/custom/icons.js
tinymce.IconManager.add('custom', {
	icons: {
		topic: '这里是自己的svg的代码',
		title: '<svg></svg>',
		quote: '<svg></svg>',
	},
})
